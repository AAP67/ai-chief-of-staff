# Changelog

All notable changes to AI Chief of Staff will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added
- Initial release
- Strategic analysis with Claude Sonnet 4
- Document processing (PDF, PPTX, XLSX, CSV, TXT, MD)
- Three analysis modes: Quick Take, Deep Dive, Scenario Analysis
- Conversation memory within session
- Export to markdown functionality
- Streamlit Cloud deployment support
- Comprehensive documentation

### Features
- AI-powered strategic decision support
- Deal and pitch deck analysis
- Scenario planning (bull/base/bear)
- Board prep assistance
- Multi-document upload and analysis

### Technical
- Built with Streamlit
- Claude Sonnet 4 integration
- Python 3.8+ support
- Environment variable configuration
- Modular architecture
